## Run the project

To clone and run teh project follow these steps:

```bash
# Clone this repository
$ git clone

# Go into the repository
$ cd styled-components-project

# Install dependencies
$ npm install

# Run the app
$ npm start
```

- React
- Styled-Components
- React-Icons
- Typescript
